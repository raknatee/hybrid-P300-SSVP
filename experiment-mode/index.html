Offline mode

we can call it like a collecting data mode

Flow program

  1. Open GUI then set the collection name and submit it before going to next step. /db
  2. Create a EEG streaming ApiV1 eeg_stream using EEG-client
  3. Click run Experiment from GUI. A client (js) would connect to backend /begin_offline_mode
  4. ML-server would store data from both GUI and EEG-client to data storage. Then, we can match signal by using timestamp.
