Offline mode

we can call it like a collecting data mode

Flow program

  1. before going to connect to WS server please check status of EEG headset from /check_headset HTTP with setInterval

  2. A client (js) connects to backend /begin_offline_mode WS
  3. ML-server would store data from both GUI and EEG-client to data storage. Then, we can match signal by using timestamp.
